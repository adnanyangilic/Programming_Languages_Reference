Returns the absolute value of the argument.

## Syntax

```py
abs(n)
```

## Example 1

Use `abs()` to return the absolute value of `-6.5`:

```cpp
#include <iostream>
#include <cmath>

int main () {
  std::cout << std::abs(-6.5) << "\n";
}

// Output: 6.5
```
