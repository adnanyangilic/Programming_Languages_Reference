Returns the inverse tangent of the argument in radians.

## Syntax

```py
atan(n)
```

## Example 1

Use `atan()` to return the inverse tangent of `0.0`:

```cpp
#include <iostream>
#include <cmath>

int main() {
  double x = 0.0;
  double result;

  result = std::atan(x);

  std::cout << result << " radians" << "\n";
  // Output: 1.5708 radians
}
```
