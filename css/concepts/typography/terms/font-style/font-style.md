To set the font style in which text will appear.

## Syntax

```css
font-style: <font-style-value>;
```

The `font-style-value` can be the following:

- `normal`
- `oblique`
- `italic`
- `inherit`
- `initial`

## Example 1

Set the `p` tag to `italic`:

```css
p {
  font-style: italic;
}
```
